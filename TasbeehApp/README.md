# TasbeehApp

تطبيق سبحة إلكترونية بسيط للأندرويد.

## المميزات
- عداد تسبيح
- اختيار الذكر
- حفظ العدد بعد قفل التطبيق
- زر إعادة العد
- اهتزاز خفيف عند الضغط

## بناء APK بدون Android Studio
1. ارفع المشروع على GitHub.
2. افتح تبويب Actions.
3. شغّل Workflow باسم Build Android APK.
4. نزّل ملف APK من Artifacts.
